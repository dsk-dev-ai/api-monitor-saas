# API Monitor SaaS

A lightweight SaaS platform to monitor APIs and websites with uptime tracking and alerts.

## Tech Stack

Frontend
- Next.js
- Tailwind CSS

Backend
- Node.js
- Express

Database
- PostgreSQL / Supabase

Worker
- Node.js monitoring service

Payments
- Stripe

Hosting
- Vercel

## Features

- API uptime monitoring
- Response time tracking
- Email alerts
- Developer dashboard
- Subscription plans

## Architecture

User → Dashboard → Backend API → Monitoring Worker → Database → Alerts
